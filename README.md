# 五子棋游戏 (Gomoku)

一个使用HTML5 Canvas开发的精美五子棋游戏。

## 功能特点

- 🎮 完整的双人对战五子棋游戏
- 🎨 精美的UI设计，带有渐变和阴影效果
- 📱 响应式设计，支持移动端
- ↩️ 悔棋功能
- 📊 计分系统
- ✨ 棋子带阴影和高光效果
- 🎯 最后落子位置高亮显示
- 🌐 可直接部署到GitHub Pages

## 游戏规则

1. 黑子先行，双方轮流落子
2. 先在横向、纵向或对角线上连成5子的一方获胜
3. 棋盘满子时为平局

## 部署到GitHub Pages

1. Fork或创建新仓库
2. 将所有文件上传到仓库
3. 在仓库设置中启用GitHub Pages
4. 选择main分支作为源
5. 等待几分钟后即可通过 `https://你的用户名.github.io/仓库名` 访问

## 本地运行

直接用浏览器打开 `index.html` 文件即可。

## 技术栈

- HTML5
- CSS3
- JavaScript (Canvas API)
